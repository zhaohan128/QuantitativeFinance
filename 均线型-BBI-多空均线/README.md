#通达信BBI-多空均线
#BBI又叫多空指数，是通过将几条不同天数的移动平均线用加权平均方法计算出的一条移动平均线的综合指标，BBI指标本身就是针对普通移动平均线MA指标的一种改进，任何事物都需要在不断推陈出新的改进中才能进步发展，技术指标也不例外。

2.股价位于BBI 下方，视为空头市场
